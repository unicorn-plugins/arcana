# 1. 프로젝트 개요

## 1.1 게임 소개 및 핵심 콘셉트

**아르카나(Arcana)**는 타로카드 기반의 2D 로그라이트 턴제 전략 게임이다. 플레이어는 운명이 정해진 세계에서 운명을 거부하고 새로운 미래를 개척하려는 주인공을 조작한다.

**핵심 콘셉트:** 불완전한 카드 드로우 속에서 덱 시너지와 전략 판단으로 매 턴 퍼즐을 해결하는 로그라이크 카드 전투 시스템

**키워드:** 로그라이트 / 전략 / 카드 / 타로 / 턴제 / 윤회

| 항목 | 내용 |
|------|------|
| 게임명 | 아르카나 (Arcana) |
| 장르 | 2D 로그라이트 턴제 전략 게임 |
| 플랫폼 | PC |
| 타겟층 | 20~30대 캐주얼 게이머, 목표 클리어의 쾌감을 좋아하는 도전적 플레이어 |

## 1.3 개발 목표 및 비전

**보스:** 아르카나의 핵심 차별점은 타로카드의 정방향/역방향 메커니즘을 전투 전략에 녹여낸 것이다. 단순한 카드 배틀이 아니라, 플레이어의 선택(운명 순응 vs 저항)이 전투 양상 자체를 바꾸는 경험을 목표로 한다.

**체리:** 시스템적으로는 빠른 전투 템포와 명확한 성장 피드백을 최우선으로 설계한다. 코어 루프 → 메타 루프의 반복에서 피로도를 최소화하면서도 "한 판만 더"의 동기를 유지하는 것이 관건이다.

**주디:** 전체적으로 사운드·비주얼·시스템이 하나의 세계관 톤 안에서 일관되게 작동해야 한다. 타로카드의 신비로운 분위기를 모든 요소에 관통시키는 것이 비전이다.

# 2. 세계관 & 스토리

## 2.1 세계관 설정 — 아르카나의 탄생과 운명의 법칙

**기획의도 (보스):** 턴제 특성과 주인공의 선택을 운명을 깨부수는 캐릭터의 행동에 몰입할 수 있는 세계관

### 시대적 배경

암흑기에서 르네상스 시대(14세기)로 인문주의가 변화되는 과정을 배경으로 한다.

### 문화적 배경

군주제이지만 종교의 힘이 강한 세계. 운명의 교단이라는 종교 중심의 세계이며, "불결한 운명을 정화한다"라는 모순적인 교리로 살생을 서슴지 않는다.

### 세계관 핵심 특징

- 사람들은 각각 자신만의 운명을 가지고 태어남
- 선천적으로 타로카드 기반의 권능(Wheel of Fortune, Hangedman 등)을 지니며, 권능의 세기는 경험에 따라 차이남
- 평상시에는 **정방향 효과**의 권능을 사용하나, 극한의 상황에서 **역방향**의 권능을 사용할 수 있음
- 인간이 아닌 크리처(판타지) 종족 또한 존재

### 배경 스토리

아르카나라는 운명의 신이 모든 것에 운명을 지정하는 질서의 세계를 창조했다. 초반에는 운명의 교단으로 하나의 목표를 위해 각각의 운명을 짊어지며 약 200년간 번창했다. 하지만 태어날 때부터 자신의 운명을 타고난다는 사실에 일부 세력들이 반발하며 운명을 거부하기 시작했다. 이로 인해 점점 많은 변수로 서서히 운명을 예측할 수 없게 되며 신앙이 약해졌다. 운명의 교단은 Emperor의 권능으로 대다수의 국민을 복종시키는 데 성공했으나, Judgment의 지시에 따라 폭력으로 진압을 시도하다 실패하여 내전이 일어난 폭력적이고 혼란스러운 상황이 펼쳐진다.

**이 세상의 주된 문제:** 운명을 거부해 자유를 찾으려는 세력과 운명으로 질서를 유지하는 세력 간의 내전

## 2.2 주요 세력 및 진영 설정

### 2.2.1 운명의 교단 (Order of Arcana)

교회와 황제가 통치하는 종교 국가이자 세계 그 자체. 오랫동안 아르카나를 통치한 집단으로, 운명에 따르는 것에 극도로 민감하다.

### 2.2.2 꿈을 쫓는 자 (Libertas)

운명을 믿지 않거나 거부하며 자신의 꿈을 위해 살고 싶어하는 집단. 자유 해방 세력.

### 2.2.3 중립 세력

양 진영 사이에서 생존을 도모하거나, 독자적 신념을 따르는 개체들.

## 2.3 주요 지역 및 배경 설정

게임은 총 4개의 스테이지로 구성되며, 각 스테이지는 운명의 교단 지배 영역을 거쳐 최종 보스 아르카나에 도달하는 여정을 그린다.

- **초반 스테이지:** 모험/자연 미학, 종교적 요소는 미묘하게, 자연이 뒤덮인 폐허 (아치, 덩굴, 석조 구조물)
- **후반 스테이지:** 종교 교단 건물 (중후한 종교적 미학), 최종 보스 페이즈에서 시계가 깨지며 별하늘이 드러남
- **메인 로비:** 실내 작전실 — 지도가 놓인 테이블, 책장, 램프 조명, 학구적 분위기

## 2.4 메인 스토리 시놉시스

주인공은 운명이 정해진 세계를 부수려는 인물이다.

**프롤로그 (튜토리얼):**
운명을 정하는 절대적 존재인 최종 보스를 물리치기 위해 모험을 떠난 주인공 일행. 하지만 최종 보스와의 전투에서 조력자(히로인)가 죽으면서 주인공은 윤회한 뒤 역방향의 힘을 각성하고, 조력자와 아군의 죽음을 막기 위해 다시 여정을 떠난다.

**상세 스토리 흐름:**

1. 운명이라는 이유로 자신의 소중한 이들이 한순간에 죽임을 당함
2. 주인공은 복수를 위해 꿈을 쫓는 자와 함께 The Emperor를 처리하러 감
3. The Emperor의 권능으로 세뇌당해 자신의 손으로 동료를 죽임
4. 조력자 Hangedman의 희생으로 세뇌에서 풀려남 — 동료들이 학살당한 상황을 마주함
5. 절망하며 역방향의 능력을 발현 → **게임 시작**
6. 스테이지를 진행하며 성장, 동료 합류
7. 아르카나를 빈사 상태로 만들었으나, 아르카나를 죽이면 운명의 권능이 모두 사라지며 조력자도 죽는다는 운명을 깨달음
8. 조력자는 주인공의 카드를 찢으면서 희생
9. 최종 전투 승리 → 조력자가 만들고 싶었던 세계를 바라보며 엔딩

## 2.5 챕터별 스토리 구성

| 스테이지 | 보스 | 스토리 테마 |
|----------|------|-------------|
| 1 | The Chariot | 진군과 결의 — 여정의 시작, 첫 시련 |
| 2 | Justice | 심판과 회의 — 정의란 무엇인가에 대한 질문 |
| 3 | The Hermit | 고독과 지혜 — 숨겨진 흑막과의 대면 |

## 2.6 엔딩 분기 설계

최종 보스 클리어 시 하나의 메인 엔딩으로 수렴한다. 주인공이 아르카나를 쓰러뜨리고 조력자가 만들고 싶었던 "정해진 운명 없는 실존주의 세계"를 만들며 마무리된다.

**스토리 컨셉 방향성 (보스):** 각 캐릭터는 개별적 스토리 및 개인의 신념을 독립적으로 가지며, 캐릭터 일부가 사라져도 어색하지 않게 설계한다. 금지된 톤으로 너무 가벼운 유치찬란한 분위기 금지, 너무 고어하고 자세한 묘사 금지.

# 3. 게임 시스템 기획

## 3.1 핵심 게임플레이 루프 (Core Loop)

**체리:** 코어 루프를 통한 전략적 성장은 전투의 전략과 맞물려 피로도가 쌓이기에, 코어 루프에서는 증강이라는 보상과 비례해 적이 강해지는 성장을 한다.

### 코어 루프

1. **윤회 시작** → 윤회 준비 (증강 선택 / 캐릭터 선택)
2. **맵(노드) 화면** → 플레이어가 이동할 노드 선택 (전투/사건/상점/보스)
3. **노드 진행** → 전투 스테이지 또는 사건 스테이지
4. **클리어 판정** → 성공 시 증강 보상 획득, 다음 노드 진행 / 실패 시 게임 오버
5. **보스 클리어** → 다음 스테이지 이동 또는 엔딩
6. **게임 오버 시** → 윤회 화면 → 토템 선택 → 다음 윤회 준비

### 메타 루프

스테이지를 진행하며 중간 중간 지정된 보스전을 준비한다. 게임 오버 시 윤회하여 첫 스테이지로 초기화되며, 전 윤회에서 얻은 능력치의 일부를 돌려받는다. 메타 루프 완료까지 예상 플레이타임은 최소 15분 ~ 최대 20분이다.

### 플레이어 목표 계층

| 시간 축 | 목표 |
|---------|------|
| 단기 (단일 런) | 전투 생존 → 다음 보상 획득 |
| 중기 (런 사이클) | 빌드 완성 → 보스 돌파 → 전략 숙련 |
| 장기 (전체 진행) | 전 스테이지 클리어 → 모든 캐릭터 해금 → 메커니즘 마스터리 |

## 3.2 장르 및 플레이 방식 정의

2D 로그라이트 턴제 카드 전략 게임. 플레이어는 카드 기반 스킬을 사용하여 전투하며, 노드 맵을 따라 선형적으로 진행한다.

**핵심 재미 요소:**
- **전략의 재미:** 카드 사용 / 턴 운영 / 시너지 / 판단
- **육성의 재미:** 캐릭터가 윤회하며 더욱 강해짐
- **인지적 재미:** 플레이어의 선택에 따라 달라지는 결과
- **쾌감:** 보스 및 어려운 도전을 클리어하는 도파민

## 3.3 운명 시스템 — 천칭 시스템 (Fate System)

### 3.3.1 천칭 시스템 정의

역방향/정방향 카드 사용에 따른 **하이 리스크 하이 리턴 / 로우 리스크 로우 리턴**의 전략적 요소를 위한 시스템이다.

- **역방향 사용 = 하이 리스크 하이 리턴 (운명 저항)**
- **정방향 사용 = 로우 리스크 로우 리턴 (운명 순응)**

**레퍼런스:** Darkest Dungeon의 정신력 시스템, No Straight Roads의 Lock 시스템

### 3.3.2 천칭 시스템 처리

1. 전투 노드 입장 시 천칭 시스템 초기화
2. 플레이어가 스킬 사용
3. 스킬 사용에 따른 증감치로 침의 기울기 실시간 변화
4. 칸에 도달 시 효과를 정해진 대상에게 즉시 적용

**스킬 카드 증감치:**
- 역방향 스킬 사용 = 운명 수치 +10
- 정방향 스킬 사용 = 운명 수치 -10

**기울기 표시:** 칸에 대한 텍스트는 비주얼적으로 표시하지 않으며, 한 칸마다 수치 50을 가진다. 스케일 범위는 -3 ~ +3 (총 7구간).

### 3.3.3 천칭 효과 — 운명 저항 (역방향 방면)

| 칸 | 효과 |
|----|------|
| -1 | 아군 전체 공격력 8% 증가 / HP 5% 감소 / 방어력 5% 감소 |
| -2 | 아군 전체 공격력 16% 증가 / HP 10% 감소 / 방어력 10% 감소 |
| -3 | 아군 전체 공격력 32% 증가 / HP 20% 감소 / 방어력 20% 감소 |

### 3.3.4 천칭 효과 — 운명 순응 (정방향 방면)

| 칸 | 효과 |
|----|------|
| 1 | 데미지 적중 시 해당 캐릭터 최대 HP의 3% 회복 / 파티 전체 공격력 3% 감소 |
| 2 | 데미지 적중 시 해당 캐릭터 최대 HP의 6% 회복 / 파티 전체 공격력 6% 감소 |
| 3 | 데미지 적중 시 해당 캐릭터 최대 HP의 12% 회복 / 파티 전체 공격력 12% 감소 |

칸마다 증감량이 2배씩 늘어나며, 상세 수치는 레벨링에서 조절 가능해야 한다.

### 3.3.5 세계관 연계 컨셉

- **운명 순응:** 더 월드가 칭찬하는 느낌 + 기본적인 적의 방해
- **운명 저항:** 아군팀이 각성하는 느낌 + 더 월드의 방해
- **중립:** 저항과 순응을 반복하며 운명 변동이 없는 안정적 플레이

## 3.4 전투 시스템

### 3.4.1 기본 전투 메커니즘

전투 노드 선택 시 진행되는 단계. 몬스터 팀이나 아군 팀 중 한쪽의 모든 캐릭터 HP가 0이 되었을 때 종료된다.

**턴 구조:**
1. 플레이어 턴 → 카드 드로우 (6장) → 스킬 사용 (드래그앤드롭) → 턴 종료 버튼
2. 몬스터 턴 → 몬스터 ID 순 행동 → 스킬 패턴 랜덤 선택 → 순차 실행 → 플레이어 턴 복귀

**스킬 코스트:** 기본 최대 코스트 10칸. 아군 첫 턴에는 코스트가 가득 찬 상태로 시작하며, 이후 아군 턴마다 3칸 회복. 미사용 코스트는 다음 턴으로 이월.

**코스트 분류:**
- 단일 대상 스킬: 코스트 1
- 복수 대상 스킬: 코스트 2
- 궁극기: 코스트 0 (소모 없음)

**HP 처리 규칙:**
- HP는 캐릭터 단위로 관리
- HP는 0 미만으로 떨어지지 않음
- HP는 최대 HP를 초과하지 않음
- 스킬 발동 후 전투불능 상태 체크

### 3.4.2 카드 드로우 및 덱 시스템

**덱 구성:**
- 아군 캐릭터 1명이 스킬 1개를 각 6장씩 보유
- 파티 3명 기준 총 18장으로 카드 덱 구성
- HP가 0이 된 캐릭터의 스킬 카드는 즉시 덱에서 제외

**드로우 규칙:**
- 타이밍: 플레이어 턴 시작 시 자동
- 드로우 수량: 18장 중 6장을 균등확률로 드로우
- 드로우 완료 후 행동 페이즈 진입

**카드 사용:**
- 조작: 드래그 앤 드롭으로 전투 화면에 올려 정/역방향 결정 후 사용
- 역방향 전환 시 코스트 소모
- 호버 시 스킬 범위 표시
- 코스트 부족 시 카드 비활성(딤 처리)

**덱 리셋 규칙:**
- 트리거: 덱에 6장 미만 남았을 때
- 재구성: 생존 캐릭터의 카드만으로 덱 재구축
- 리셋 후: 정상 드로우 규칙 적용
- 턴 종료 시: 미사용 카드는 덱으로 복귀

### 3.4.3 시너지 시스템

시너지는 같은 종류의 카드를 연속 사용할 때 발동되는 누적 공격력 증가 시스템이다.

**발동 조건:** (이전 카드 종류) == (현재 카드 종류)

**스택 효과:**
- 스택 1: 기본 데미지 (시너지 시작점)
- 스택 2: 공격력 +n%
- 스택 3: 공격력 +2n%
- 스택 4: 공격력 +3n% (풀 스택)

**카드 분류:** 정방향 마이너 카드 / 정방향 메이저 카드 / 역방향 메이저 카드(궁극기)

**시너지 유지:** 같은 종류 카드 또는 특수 카드 사용 시 유지. 특수 카드는 종류 무관하게 시너지 효과를 받음.

**시너지 초기화:**
- 종류 불일치 시 (마이너→메이저 등) 스택 리셋
- 궁극기 사용 시: 누적 시너지를 보너스로 소비 후 전체 스택 리셋
- 턴 종료 시: 모든 스택 소멸

**풀 스택 궁극기 보너스 (안):**
- 화력 증폭: 현재 시너지 보너스 데미지 2배
- 관통 효과: 적 방어력 고정량 무시

### 3.4.4 궁극기 시스템

**아군 궁극기:**
- 한 캐릭터의 스킬을 3회 연속 사용하면 궁극기가 카드 형태로 손 덱에 진입
- 궁극기 사용 시 미사용 캐릭터에게 궁극기 스택 1씩 부여
- 일정 스택 충족 시 해당 캐릭터의 궁극기도 손 덱에 진입
- 스택으로 얻은 궁극기는 사용해도 스택을 부여하지 않음

**적군 궁극기:**
- 보스 캐릭터의 사용 완료된 스킬 횟수를 집계하여 일정 횟수 충족 시 궁극기 활성화
- 활성화 즉시 발동, 사용 후 다시 횟수 집계 시작

### 3.4.5 특수 카드 시스템

**획득:** 잡 몬스터 HP가 0이 되면 해당 몬스터의 스킬 중 랜덤 1장 드롭

**특성:**
- 코스트: 0 (무료)
- 유효 범위: 해당 노드 웨이브 내에서만 유효
- 웨이브 전환 시 리셋
- 턴 종료 시 미사용 특수 카드 자동 소멸
- 웨이브당 최대 획득: 2장 (전투당 최대 적 3체)
- 시너지: 종류 무관하게 시너지를 이어나갈 수 있음

## 3.9 캐릭터 성장 시스템

### 3.9.1 캐릭터 스탯 구조

캐주얼 게임 특성에 맞게 이해하기 쉬운 형태로 고정한다:
- **공격력 (DMG)** — 공격, 버프, 디버프 관련 스킬에 영향
- **방어력 (DEF)** — 방어, 버프, 디버프 관련 스킬에 영향
- **HP** — 힐, 버프, 디버프 관련 스킬에 영향

### 3.9.2 증강 시스템

전투 노드를 승리하면 확정적으로 등장하는 성장 단계.

**호출 시점:**
- 전투 노드 클리어: 1회 호출 (3장 중 1장 선택)
- 보스 노드 클리어: 2회 호출 (각 3장 중 1장 선택)

**카드 생성 규칙:** 노드 타입에 맞는 증강 테이블에서 무작위로 3장 추출하여 UI 배치. 한 화면에 슬롯 ID + 증강 효과 + 수치가 완전히 동일한 카드는 등장 불가. 증강 효과가 같더라도 대상(개별 슬롯/전체 슬롯)이 다르면 별개 카드로 동시 등장 가능.

**증강 효과 타입:**

| 타입 | 설명 | 연산 방식 |
|------|------|-----------|
| 스택 증가 (고정치) | 공격력/방어력/HP 고정치 증가 | 합 연산: 기본값 + Σ고정치 |
| 비율 증가 | 스킬 적용량 배율, 범위 증가 등 | 합산 후 일회 적용: 기본값 × (1 + Σ비율) |
| 리스크 트레이드 | 방어력 -N, 공격력 +N 등 | 개별 합 연산 |

**스테이지별 스케일링:**
- 스탯 증강: 스테이지에 따라 수치 증가 (1스테이지: +100, 2스테이지: +200, 3스테이지: +300)
- 스킬 증강: 비율 기반이므로 스테이지 영향 없음

**상한선:**
- 깡 스탯 (공격력/HP): 상한선 없음 (성장의 재미를 위해)
- 공격 관련 비율: 최대 +n% 제한
- 방어 관련 비율: 최대 +n% 제한
- 스킬 코스트 감소: 코스트 1 이하로 감소 불가
- 스킬 범위: 최대 적 수(3체) 초과 불가

**증강은 슬롯에 귀속 (캐릭터가 아님):** 캐릭터를 교체해도 슬롯의 증강은 유지된다.

### 3.9.3 캐릭터 토템화 시스템

아군 팀 전체의 HP가 0이 되면 패배 판정 이후 캐릭터 토템화 단계로 진입한다.

- 패배 판정 당시 아군 캐릭터 중 하나를 선택해 토템으로 생성
- 선택한 캐릭터의 가장 높은 수치 특성이 윤회 후 첫 노드부터 패시브 버프로 적용
- 토템은 턴·노드·윤회가 넘어가도 사라지지 않으며, 플레이어가 삭제 불가
- 예시: 공격력이 가장 높은 캐릭터 → 아군 팀 공격력 버프 토템
- 최고 수치 동률 시 우선순위 적용

## 3.10 노드 시스템 및 탐색

### 3.10.1 스테이지 구성

게임은 최종 보스까지 3개의 스테이지로 구성 (노드시스템V5 기준). 한 스테이지는 6개의 레이어로 구성된다:

| 레이어 | 타입 | 개수 | 비고 |
|--------|------|------|------|
| 레이어 1 | 전투 (고정) | 1 | 시작 노드 |
| 레이어 2 | 랜덤 (전투/사건) | 1~3 | 절차 생성 |
| 레이어 3 | 랜덤 (전투/사건) | 1~3 | 절차 생성 |
| 레이어 4 | 랜덤 (전투/사건) | 1~3 | 절차 생성 |
| 레이어 5 | 상점 (고정) | 1 | 보스 전 상점 |
| 레이어 6 | 보스 (고정) | 1 | 스테이지 보스 |

### 3.10.2 노드 맵 구조

- 6개 고정 레이어 내에서 절차적으로 경로 생성
- 플레이어는 연결된 다음 레이어 노드 중 하나를 선택하여 진행
- 이전 레이어로 복귀 불가 (단방향 진행)
- 네트워크 구조로 시각 표현 (색상 연결선, 아이콘 표시 노드)

### 3.10.3 노드 진행 흐름

1. **노드 생성:** 레이어별 조건에 따라 절차적 생성
2. **노드 선택:** 플레이어가 연결된 다음 레이어 노드 선택
3. **파티 편성 (전투/보스 노드):** 전투 진입 전 파티 편성 창 활성화
4. **노드 실행:** 선택한 노드 타입에 따라 진행
5. **증강 & 전진:** 노드 클리어 시 증강 선택 후 다음 노드 활성화

**진행 규칙:**
- 현재 노드를 클리어해야 다음 노드 활성화
- 증강 선택 완료 전까지 다음 노드 비활성
- 현재 노드 하이라이트, 보스 노드 시각적 차별화

### 3.10.4 전투 노드

각 스테이지 시작 시 보스의 종류가 정해지고, 보스에 대한 힌트를 텍스트나 컷신으로 은유적으로 제공. 전투 노드 클리어 시 증강을 받고 다음 노드로 진행.

### 3.10.5 사건 노드

랜덤 레이어(2~4)에서 확률적으로 등장. 전투 노드와 사건 노드 중 선택 가능. 사건 노드 완료 시 증강 선택 없음.

### 3.10.6 상점 노드

레이어 5에 고정 배치. 장비·카드·소모품 구매 가능.

## 3.11 사건 시스템

**기획의도 (보스):** 리스크를 명확히 전달하고, 순수 손해 결과를 피하며, RNG에 의한 처벌을 배제하고, 플레이어의 어떤 선택이든 '정답'이 되도록 설계한다.

**사건 목록 (10종):**

| # | 사건명 | 상황 | 선택지/보상 |
|---|--------|------|------------|
| 1 | 붕괴 (The Tower) | 건물이 무너지려 한다 | 훔침 (최대HP -N, 증강+N, 캐릭터 사망) / 사람 구함 (최대HP -N, 이벤트 1-1 진입) / 대피 (코스트 +N) |
| 1-1 | 붕괴 후속 | 구한 사람이 보상 제안 | 증강 교환 (+N) / 처치 후 탈취 (증강+N, 사망) / 캐릭터 영입 (캐릭터 보상) |
| 2 | 도망치는 부상자 | 운명 교단 복장의 부상자 | 직접 처치 (캐릭터 증강+N) / 별을 쫓는 자에 인계 (증강+N) / 도움 (코스트+N) |
| 3 | 변화의 문 | 폐허 속 검은 문 | 들어감 (캐릭터 보상, 전체 증강+N) / 닫음 (캐릭터 증강+N) |
| 4 | 반가운 지원군 | 별을 쫓는 자 무리 조우 | 물자 제공 (다음 전투 적 약화) / 거절 (코스트+N) |
| 5 | 수상한 야영지 | 모닥불과 스프 | 마심 (캐릭터 증강+N) / 독 투입 (다음 전투 적 약화) / 불 키움 (HP-N, 적 약화) |
| 6 | 균형의 샘 | 질서의 샘 | 전원 입수 (캐릭터 증강+N) / 1명 입수 (증강+N) / 물 담아감 (코스트+N) |
| 7 | 왜곡된 심판 | 배신자 심판 장면 | 무죄 선언 (HP-N, 전체 증강+N) / 처리 (캐릭터 증강+N) / 천칭 기울기 (코스트+N) |
| 8 | 떨어진 별 | 밝은 별이 떨어져 빛남 | 빛 따라감 (코스트+N) / 여기서 쉼 (전체 증강+N) |
| 9 | 끊이지 않는 행군 | 먼 곳에서 군대가 행진 | 행군 합류 (스킬 카드 출현율 선택) / 식량에 독 (다음 전투 적 HP/방어-N%) |

## 3.12 파티 시스템

### 파티 구성 규칙

- 한 파티는 최대 **3종의 캐릭터**로 구성
- 최소 1명 이상 편성되어야 전투 노드 진입 가능
- 파티는 총 3개의 독립된 **슬롯**으로 구성
- 동일 캐릭터 카드 중복 배치 불가
- 획득한 증강 수치는 캐릭터가 아닌 **해당 슬롯에 영구 귀속**

### 파티 설정창 호출 시점

- 전투/보스 노드 진입 시: 맵에서 노드 클릭 시 즉시 호출
- 메인 메뉴: 전투 외 상황에서 상시 정비용 호출 가능

### 패시브 시스템

모든 아군 캐릭터는 패시브를 보유. 파티에 포함되면 자동 적용되며, 서로 유기적으로 연결될 수 있도록 설계한다.

예시:
- 아군 1: 힐을 받으면 아군 방어력 증가
- 아군 2: 스킬 사용 시 확률적으로 아군 힐
- 아군 3: 확률적으로 발동되는 패시브의 확률을 올림
- 아군 4: 방어력이 증가하면 자힐

## 3.13 보스 페이즈 시스템

**기획의도 (보스):** 단일 전략으로 보스를 클리어할 수 없도록 페이즈 전환으로 전투 다양성을 부여하고, 적응 성공 시 충분한 보상으로 쾌감을 제공한다.

**페이즈 제한:**
- 중간보스: 최대 2페이즈
- 최종보스: 최대 3페이즈

**페이즈 전환 조건:** HP 약 50% 도달 시

**페이즈별 구조:**

| 페이즈 | 적용 | 특징 |
|--------|------|------|
| 1 (기본) | 전 보스 | 초기 상태, 스킬 A·B로 기본 패턴 |
| 2 (기믹 발동) | 전 보스 | HP ~50%에서 전환, 궁극기 1회 보장 사용 |
| 3 (최종) | 최종보스만 | HP ~50%에서 전환, 즉사 패턴 궁극기 3턴마다 발동 |

**최종보스 특수 기믹:** 페이즈 2 진입 시 The Emperor → The World로 HP 바 교체 (컷신 연출)

## 3.14 전투 패턴 시스템

### 잡 몬스터 패턴

- 실행 순서: 몬스터 ID 순으로 턴 부여
- 공격 구성: 1개 스킬 패턴 선택 후 순차 실행
- 스킬 패턴: 각 몬스터는 마이너 아르카나 2종 보유, 순서 있는 조합
- 패턴 예시: [A], [B], [A,A], [A,B], [B,A], [B,B]

**타게팅 패턴 (4종):**

| 타입 | 규칙 | 효과 |
|------|------|------|
| 번호 매칭 | 몬스터 ID = 캐릭터 ID | 단순·반복적 공격 구조 |

| 마지막 공격자 | 직전 턴에 자신을 공격한 캐릭터 | 자주 사용하는 캐릭터 억제 |
| 랜덤 | 무작위 (중복 허용) | 예측 불가 |

### 보스 패턴

**중간보스 스킬 구성:** A (단일), B (전체) + 궁극기
**최종보스 스킬 구성:** A (단일), B (단일), C (전체), D (전체) + 궁극기

**최종보스 패턴:** 2~3개 스킬 조합, 순서 있음, 중복 없음
- 예시: [A,B], [A,C], [A,D], [B,C], [B,D], [C,D], [A,B,C], [A,C,D] 등

## 4.1 주인공 — Wheel of Fortune

| 항목 | 내용 |

## 4.2 주요 동료 캐릭터

### Justice — 방어형 후반 캐릭터

|------|------|

### 중간보스 — The Emperor (스테이지 4 전)

|------|------|

| 이리오너라 | 중간 보스 N명 소환, 소멸 시 시전자 방어력 약화 |
| 궁극기 — 절대 통제 | 적을 일정 턴 아군 전향, 전향 해제 시 시전자 디버프 |

### 최종보스 — The World (아르카나)

아르카나 세계 그 자체. 모든 윤회를 관장하는 존재.

**3페이즈 구성:**
- **1페이즈:** 스킬 A, B (단일 2개)
- **2페이즈:** The Emperor에서 전환, HP 바 교체. 스킬 A, C, D (단일 1개, 다중 2개)
- **3페이즈:** 스킬 A, B, C, D + 즉사 패턴 궁극기 (3턴마다 발동)

| 스킬 | 효과 |
|------|------|
| 운명의 고리 | 단일 공격 + 방어 디버프 |
| 운명 재배열 | 단일 지정, 일정 턴 동안 윤회 초기 스탯으로 변경 |
| 예정된 운명 | 중간 보스 N명 소환, 소멸 시 시전자 방어력 약화 |
| 궁극기 — 완성된 운명 | 양 팀 각 1캐릭터에 즉사 디버프. 지정 턴 내 해제 실패 시 즉사. 2회차 주인공은 불사 |

## 4.5 잡 몬스터

각 몬스터는 마이너 아르카나 기반의 스킬을 지닌다. 비인간형 외형으로 사물(컵, 검, 지팡이, 오망성)에 빙의한 형태. 유령 같은 인간 실루엣(눈, 손)이 사물 안에 드러남.

| 마이너 | 속성 | 스킬 예시 |
|--------|------|-----------|
| 오망성 (Pentacles) | 버프, 회복 | 단일 회복 |
| 검 (Swords) | 데미지, 공격력 디버프 | 광역 데미지 |
| 컵 (Cups) | 데미지, 회복 디버프 | 단일 데미지 + 방어 디버프 |
| 지팡이 (Wands) | 데미지, 방어력 디버프 | 단일 공격 + 디버프 수 추가 데미지 |

잡 몬스터는 정방향 스킬 2개를 보유. 순서 관계 있는 1~2개 스킬 세트를 6가지 패턴으로 운용. 처치 시 특수 카드 드롭.

# 5. 밸런싱 설계

## 5.1 밸런싱 철학 및 기준

**보스:** 코어 루프에서는 증강이라는 보상과 비례해 적이 강해지는 성장 구조를 취한다. 전투에서의 전략과 맞물려 플레이어 피로도가 쌓이지 않도록 성장 피드백을 명확히 설계한다.

**핵심 원칙:**
- 전략이 운(RNG)을 이길 수 있어야 함
- 리소스(코스트)는 항상 부족하게 느껴져야 함
- 도전 클리어 시 도파민 보상

## 5.2 전투 수치 설계

캐주얼 게임 특성에 맞게 공격력/방어력/HP 3종의 스탯만 사용. 스탯(깡 수치)은 성장의 재미를 위해 별도 상한선을 두지 않으나, 비율 증가에 대해서는 상한을 적용한다.

- 공격 관련 비율: 최대 +n% 제한
- 방어 관련 비율: 최대 +n% 제한
- 스탯 관련 비율: 최대 +n% 제한

## 6.2 캐릭터 아트 가이드

### 6.2.1 카드 일러스트 vs 인게임 스프라이트 규격

**카드 일러스트:**
- 카드 구성: 배경 + 캐릭터 + 외각 프레임 + 하단 이름표 (레이어 구조)

## 8.2 전투 액션 모션

**궁극기 애니메이션 구조:** 4컷 (진입, 전개, 클라이맥스, 퇴장)
- 진입/퇴장: 3콤마 (8f/s, 약 1초)
- 전개: 2~3콤마 (8~12f/s)
- 클라이맥스: 1콤마 (24f 풀 홀드)

**포춘(주인공):**
- 정방향: 공중부양 + 왼팔을 위로 뻗으며 발을 앞으로, 수레바퀴가 손 앞으로 나오면서 링 회전
- 역방향: 오른손을 얼굴까지 올리고 대기 → 빠르게 핑거스냅 → 수레바퀴 투척
- 궁극기: 수레바퀴와 함께 대규모 회복 이펙트

**저스티스:**
- 정방향: 가장 왼쪽 몬스터 기준 순간이동 → 검을 크게 휘두르는 광역 모션
- 역방향: Idle에서 검끝을 돌리며 역수로 쥠 → 반격 시 검을 위로 쳐올림

**매지션:**
- 정방향: 점프 → 팔을 양쪽으로 펼침 → 핑거스냅 → 마술사 엔딩 포즈
- 역방향: 점프하며 채찍 휘날림 → 양손으로 채찍 끝 잡아 당김 → 크게 휘둘러 공격

**더 풀:**
- 정방향: 점프 → 팔 양쪽 펼침 → 최고 높이에서 핑거스냅
- 역방향: 점프하며 한쪽 다리 크게 올림 → 채찍으로 광역 공격

| 용어 | 설명 |
|------|------|
| 노드 | 플레이어가 전투·사건을 진행하는 단위 |
| 윤회 | 게임 오버 후 첫 스테이지 첫 노드부터 재시작하는 행위 |
| 사건 | 확률적으로 등장하는 캐릭터 성장·변화 선택지 노드 |
| 스테이지 | 전체 노드의 집합 |
| 아군턴 / 적군턴 | 전투 중 행동 가능한 턴 |
| 잡 몬스터 | 중간/최종 보스를 제외한 몬스터 |
| 토템 | 윤회 시 다음 윤회에 이점을 주기 위해 남기는 버프 |
| 드로우 | 카드 덱에서 카드를 랜덤으로 뽑는 행위 |
| 시너지 | 같은 종류 카드 연속 사용 시 발생하는 누적 공격력 증가 |
| 전투불능 | HP 0으로 전투 참여 불가 상태 |
| 특수 카드 | 잡 몬스터 처치 시 드롭되는 코스트 0 카드 |
| 정방향 | 기본 스킬 상태 (운명 순응) |
| 역방향 | 코스트 소비하여 전환한 강화 상태 (운명 저항) |
| 궁극기 스택 | 아군 궁극기 사용 시 미사용 캐릭터에 쌓이는 수치 |
| 드롭 | 몬스터 전투불능 시 특수 카드를 덱에 넣는 행위 |
| 증강 | 전투/보스 클리어 후 획득하는 영구 스탯/스킬 강화 |
| 슬롯 | 파티 편성 위치. 증강은 슬롯에 귀속 |
| 웨이브 | 전투 노드 내 적 그룹 단위 |
| 페이즈 | 보스 HP에 따른 전투 패턴 전환 단계 |
| 마이너 아르카나 | 잡 몬스터가 사용하는 4종 속성 (검/컵/지팡이/오망성) |
| 메이저 아르카나 | 주요 캐릭터가 보유한 타로 기반 권능 |

# Help

[Help 스킬 활성화]

## 목표

arcana 플러그인의 모든 스킬 목록과 사용법을 안내한다. 18개 스킬(기본 5개 + 역할별 13개)의 명령어, 설명, 자동 라우팅 키워드를 하드코딩된 목록으로 표시한다.

## MUST 규칙

| # | 규칙 |
|---|------|
| 1 | 스킬 목록은 이 파일에 하드코딩된 내용을 그대로 출력한다 |
| 2 | 기본 스킬 5개와 역할별 스킬 13개, 총 18개를 모두 안내한다 |
| 3 | 각 스킬의 명령어, 역할 설명, 자동 라우팅 키워드를 함께 안내한다 |
| 4 | 현재 프로젝트 컨텍스트(프로젝트명, 현재 페이즈, 팀 구성)를 함께 안내한다 |
| 5 | 자연어 입력 시 core 스킬이 자동으로 라우팅됨을 안내한다 |

## MUST NOT 규칙

| # | 금지 사항 |
|---|----------|
| 1 | 런타임에 디렉토리를 스캔하여 스킬 목록을 동적으로 생성하는 것 |
| 2 | 파일 시스템을 탐색하여 SKILL.md 파일을 읽어 목록을 구성하는 것 |
| 3 | 스킬 목록을 외부 소스에서 로드하는 것 |
| 4 | 안내 없이 다른 스킬을 직접 호출하는 것 |

## 검증 체크리스트

- [ ] 기본 스킬 5개가 모두 안내되었는가
- [ ] 역할별 스킬 13개가 모두 안내되었는가
- [ ] 각 스킬의 명령어와 자동 라우팅 키워드가 포함되었는가
- [ ] 자연어 라우팅 사용법이 안내되었는가
- [ ] 현재 프로젝트 컨텍스트가 안내되었는가

## 자동 라우팅 (추천)

슬래시 명령어 없이 **자연어로 입력**하면 core 스킬이 자동으로 적절한 역할 에이전트로 라우팅합니다.

**예시:**
- "천칭 수치 밸런싱 검토해줘" → `/arcana:balance` 자동 라우팅
- "HUD 와이어프레임 설계해줘" → `/arcana:uiux` 자동 라우팅
- "포춘 캐릭터 전투 모션 스펙 작성해줘" → `/arcana:animation` 자동 라우팅

## 핵심 게임 시스템 (도메인 컨텍스트)

- **천칭 시스템**: 정방향/역방향 카드 사용에 따른 하이리스크-하이리턴 / 로우리스크-로우리턴
- **카드 시너지**: 정방향 세트, 역방향 세트, 같은 스킬 다른 방향 세트
- **증강 시스템**: 합연산(깡스탯) + 비율연산(배율) 누적, 노드 클리어 보상
- **궁극기**: 3회 연속 사용 발동, 스택 기반 전파
- **토템/윤회**: 패배 시 캐릭터 토템화, 다음 윤회에 패시브 버프 이전
- **노드 맵**: 4스테이지 × 4노드(전투/사건/보스), 선형 진행
- **파티 시스템**: 3슬롯, 증강은 슬롯 귀속

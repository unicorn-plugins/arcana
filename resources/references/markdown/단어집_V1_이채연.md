# 단어집_V1_이채연

## 단어집

|  | 단어  | 설명 | 사용 예시(실제 플레이와 다를 수 있음) |
| --- | --- | --- | --- |
|  | 노드 | 플레이어가 전투, 사건을 진행하는 노드 개념 | 첫 노드에서 전투에서 승리한 플레이어는 보상을 획득한다.  |
|  | 윤회 | 플레이어가 게임 오버가 되고 나서 다시 첫 스테이지의 첫 노드부터 시작하는 행위를 의미.  | 게임 오버 된 플레이어 캐릭터들이 윤회하여 첫 노드부터 다시 플레이를 시작한다.  |
|  | 윤회 N턴째 | 윤회를 반복한 횟수를 의미.  | 윤회 3턴 째에 강력한 토템을 얻어서 기쁘다~ |
|  | 사건 | 노드를 선택할 때 확률적으로 등장하는 층 캐릭터를 성장, 변화 시킬 수 있는 선택지가 등장한다. 사건도 하나의 노드으로 계산. | 전투가 끝나고 운이 좋아 등장한 사건 노드를 선택했다.  |
|  | 스테이지 | 전체 노드의 집합을 의미.   | 스테이지를 완료하고 엔딩을 맞이한다.  |
|  | 아군턴 | 전투 중 플레이어가 행동 할 수 있는 턴을 의미한다.  | 아군턴에 스킬을 사용하여 적을 한번에 물리쳤다.  |
|  | 적군턴 | 전투 중 몬스터가 행동 할 수 있는 턴을 의미한다.  | 적군턴에 몬스터가 사용한 스킬 때문에 아군이 피해를 입었다.  |
|  | 잡 몬스터 | 중간 보스와 최종 보스를 제외한 자잘한 몬스터 | 잡 몬스터를 소환하는 패턴이 까다롭다.  |
|  | 중간 보스 | 4번째 노드, 8번째 노드에 등장하는 보스 몬스터 | 4번째 노드의 중간 보스를 물리치고 큰 보상을 얻었다.  |
|  | 최종 보스 | 12층(최종보스 노드)에 등장하는 보스 몬스터 | 마지막 노드의 최종 보스를 물리치기 위해 충분한 성장을 이루었다.  |
|  | 토템  | 플레이어 캐릭터가 윤회할 때 다음 윤회의 캐릭터에게 이점을 주기 위해 남기는 능력치, 버프 등을 의미.  | 토템 2개로는 데미지가 안나오는 것 같아서 의도적으로 게임 오버 하여 토템을 다시 얻어서 전투에 대비하기로 했다.  |
|  | 드로우 | 카드 덱에서 카드를 랜덤으로 뽑는 행위  | 턴이 시작할 때 드로우로 새로 카드를 뽑았다.  |
|  | 시너지 | 사용한 스킬들의 조합에 따라 발생하는 이로운 효과 | 스킬을 사용할 때는 시너지를 고려해서 사용한다.  |
|  | 전투불능 | 개별 HP를 지닌 객체가 HP가 0이 되었을 때 더 이상 전투에 참여하지 못하게 되는 상태를 의미한다.  | 아군 중 한명이 전투 불능 상태에 빠져서 해당 아군이 가지고 있는 스킬 카드가 드로우 되지 않는다.  |
|  | 특수 카드 | 잡 몬스터를 전투 불능 상태로 만들었을 때 드랍되는 카드. 궁극기 조건이나 시너지를 이어나갈 때 사용할 수 있다. 사용 코스트가 0이다  | 잡 몬스터를 먼저 전투 불능 상태로 만들어 특수 카드를 얻고 시너지를 이어 나갔다.   |
|  | 궁극기 (적군) | 적군이 스킬을 일정 횟수 사용하여 충전할 수 있는 스킬, 다른 일반 스킬보다 강력한 효과를 가진다.  | 보스가 궁극기를 사용하여 아군팀에게 큰 피해를 주었다. |
|  | 정방향 | 아군 캐릭터의 기본 스킬 상태 | 스킬 활용을 잘 해야 클리어가 편하다.  |
|  | 역방향 | 플레이어가 코스트를 소비하여 캐릭터의 스킬을 돌렸을 때 변화한 상태 | 코스트를 소비하여 역방향으로 만든 스킬은 이 카드의 원래 형태인 정방향보다 조금 더 강하다.  |
|  | 궁극기 스택 | 아군 캐릭터가 궁극기를 사용했을 때 궁극기를 사용하지 않은 아군들에게 쌓이는 것.  | 저스티스가 궁극기를 사용해서 주인공과 매지션의 궁극기 스택이 쌓였다 |
|  | 궁극기(아군) | 한 캐릭터의 스킬을 3회 연속으로 사용했을 때, 궁극기 스택이 다 쌓였을 때 손에 든 덱으로 들어오는 강력한 스킬 | 저스티스의 스킬을 3회 연속으로 사용해 궁극기 카드를 얻었다.  |
|  | 팀 | 개체의 묶음을 이야기한다.  | 아군팀에 있는 캐릭터를 잘 살펴보기 위해 클릭해서 보유 하고 있는 스킬을 확인했다.  |
|  | 드랍 | 잡 몬스터를 전투 불능으로 만들었을 때 특수 카드를 플레이어 덱으로 넣는 행위  | 드랍된 마이너 특수 카드로 시너지를 수월하게  이어나갔다.  |
|  | 전투 필드 | 캐릭터들이 올라가있는 공간 (중경)을 의미.  | 전투 필드에 있는 캐릭터 위로 스킬 카드를 드래그 드랍해서 사용했다.  |
